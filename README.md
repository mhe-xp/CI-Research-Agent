# 计算成像科研智能体助手

## 项目简介

本项目实现了一个针对计算成像的科研智能体助手。该系统基于 ReAct (Reasoning and Acting) 范式构建，旨在协助科研人员完成从复杂光学论文分析到物理仿真、模型训练代码生成与调试的完整工作流，提升工作效率。

## 技术栈

1. 核心框架：LangChain
2. 大模型引擎：Gemini 3
3. 执行环境：PyTorch (深度学习), Python/MATLAB (光学仿真)
4. 知识库：ChromaDB (向量数据库)

## Computational Imaging Research Agent Assistant

## Project Overview

This project implements a research agent assistant specialised for computational imaging. Built upon the ReAct (Reasoning and Acting) paradigm, the system aims to assist researchers throughout the entire workflow—from analysing complex optical papers to physical simulations, model training, code generation, and debugging—thereby enhancing operational efficiency.

## Technology Stack

1. Core Framework: LangChain
2. LLM: Gemini 3
3. Environment: PyTorch (Deep Learning), Python/MATLAB (Optical Simulation)
4. Database: ChromaDB (Vector Database)
